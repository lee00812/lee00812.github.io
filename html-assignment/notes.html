5 HTML Tags I Have Learnt

Lines Tag ‹hr›
This tag provides a horizontal line to split sections on the web page.
This is an open tag.
Ordered List ‹ol› and ‹/ol›
These tags allow you to write a list and listing all items in order.
To add list items, adding the open and close tags of ‹li› and ‹/li› are necessary.
Article ‹article› and ‹/article›
Adding the acticle tag when you add an acticle or paragraph.
This tag could give us a clear view on the codes and easy to locate and identify what the content is.
Adding Images ‹img›
This tags allows you to add an image on the page.
There are two elements which are neceesary to add in this tag: they are src and alt.
Adding src allows you to add the image path.
Adding alt allows you to add an alt text for the image.
This is an open tag.
Line break ‹br›
Line break tag is one of the important tags when you write an article, marketing message, address or anything that needs an line break.
